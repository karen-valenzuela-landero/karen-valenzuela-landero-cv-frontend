# 💡 Project Overview

This is a personal portfolio frontend application built with Next.js and TypeScript, designed to display dynamic developer profile data fetched from a REST API.

The UI is styled using Tailwind CSS and enhanced with smooth animations using Framer Motion. It follows a modern responsive design approach and is fully production-ready.

# ⚙️ Tech Stack

Next.js
TypeScript
Tailwind CSS
Framer Motion
REST API integration

# 🔗 Features

Dynamic profile data from backend API
Responsive modern UI
Animated components with Framer Motion
Clean component-based architecture
API documentation integration ([Swagger ](https://karen-valenzuela-landero-cv.onrender.com/swagger-ui.html))
